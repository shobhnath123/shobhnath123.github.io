Responsive Design Test

This page helps you test the responsive design of your portfolio website.

Current Screen Size:

How to Test Responsive Design:

  1. Desktop Testing: Open your portfolio in a browser and resize the window
  2. Mobile Testing: Use browser DevTools (F12) → Toggle Device Toolbar
  3. Real Device Testing: Test on actual phones and tablets
  4. Orientation Testing: Rotate devices to test landscape/portrait modes

Responsive Breakpoints

Your portfolio uses these breakpoints for optimal display:

Optimized for tablets and small laptops

Tablets

Small Mobile

max-width: 480px

Extra Small

Minimal spacing, essential content

Key Responsive Features

  • Mobile Navigation: Hamburger menu with smooth animations
  • Flexible Grids: Auto-adjusting layouts for all screen sizes
  • Touch-Friendly: 44px minimum touch targets
  • Optimized Typography: Readable text at all sizes
  • Responsive Images: Properly scaled avatars and icons
  • Form Optimization: Mobile-friendly input fields
  • Performance: Reduced animations on mobile

Testing Checklist

  • ✅ Navigation menu collapses on mobile
  • ✅ All sections stack properly on small screens
  • ✅ Text remains readable at all sizes
  • ✅ Buttons and links are touch-friendly
  • ✅ Forms work well on mobile devices
  • ✅ Images scale appropriately
  • ✅ No horizontal scrolling on mobile
  • ✅ Smooth transitions and animations

Browser DevTools Testing

Use these device presets in Chrome DevTools:

  • iPhone SE: 375x667 (small mobile)
  • iPhone 12 Pro: 390x844 (medium mobile)
  • iPad: 768x1024 (tablet portrait)
  • iPad Pro: 1024x1366 (large tablet)
